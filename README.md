# DTERP Boat Bobbing

A lightweight FiveM script that prevents boats from drifting when the engine is off.
<img width="902" height="26" alt="image" src="https://github.com/user-attachments/assets/5de3d443-ad91-4eb9-93a1-ca2d9981145c" />

## Features
✔ Low CPU usage  
✔ Boat-only detection  
✔ Driver-only checks  
✔ Optimized loop timing  

## Installation

1. Download the latest release
2. Place the folder in your `resources` directory
3. Add to your `server.cfg`

ensure dterp-boat-bobbing

## Requirements

None (Standalone)
